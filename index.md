### [IS 6850 Home Credit Default Risk Project 🏡](https://github.com/owen-simon/IS-6850-home-credit-project) 

Graduate capstone project completed in the MSBA program at the University of Utah using data from the [Kaggle Home Credit Default Risk competition](https://www.kaggle.com/competitions/home-credit-default-risk/data).

#### Project Summary

Built an end to end credit default prediction pipeline by cleaning, integrating, and engineering features across multiple relational datasets. Transformed raw application and loan history data into consolidated train and test datasets optimized for predictive modeling.

**Key contributions:**

- Cleaned and standardized application data, handled anomalies, and implemented robust missing value strategies
- Engineered financial ratios, demographic transformations, and interaction features informed by EDA
- Aggregated bureau, POS, credit card, installment, and previous application data into client level metrics
- Ensured strict train and test separation when computing medians, bins, and derived features
- Developed modular, reusable R functions for cleaning, aggregation, and feature engineering

#### Modeling

Implemented and compared Logistic Regression, LASSO, and Random Forest models using cross validated ROC AUC.

- Selected LASSO as the final model due to strong CV performance, generalizability, and interpretability
- Conducted threshold analysis to optimize expected profit and approval strategy
- Structured code for reproducible training, validation, and evaluation workflows

#### Model Governance and Business Framing

Developed a stakeholder focused model card documenting:
- Business context and decision strategy
- Performance metrics and benchmark comparisons
- Feature importance and interpretability
- Fairness and approval rate analysis across demographic groups
- Deployment considerations and regulatory constraints
- Repository Highlights
- Quarto notebooks for EDA, modeling, and model documentation
- Reusable feature engineering pipeline in R
- Formal business problem statement
- Rendered HTML outputs for easy review

This project demonstrates strong capabilities in data engineering, feature construction, predictive modeling, model evaluation, and responsible AI documentation within a financial risk context.
